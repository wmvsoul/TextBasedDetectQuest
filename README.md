# Текстовый детективный квест «Смертельный антракт»
# Лабораторная работа — Разработка интерактивного приложения Windows Forms


## Описание проекта
Текстовый детективный квест «Смертельный антракт» — это интерактивная игра жанра детектив, в которой игроку предстоит расследовать убийство театрального режиссёра. Игра реализована на платформе Windows Forms с использованием языка C#.

Сюжет: В театре «Зеркало сцены» убит главный режиссёр — Константин Верховский. Игроку предстоит собрать улики, допросить подозреваемых и найти настоящего убийцу.

## Как запустить
Через Visual Studio
1. Откройте файл TextBasedDetectQuest.sln или TextBasedDetectQuest.csproj
2. Нажмите Ctrl + F5 для запуска без отладки или нажмите F5 для запуска с отладкой

## Пример использования программы
1. Вступление
При запуске отображается стартовое окно с описанием жанра текстовых квестов и краткой аннотацией к игре.

2. Создание персонажа
Игрок выбирает:
* Имя персонажа (до 10 символов)
* Роль: Частный детектив или Журналист
* Черта характера: Харизматичный, Циничный, Наблюдательный, Эмпатичный, Бесстрашный, Педантичный
* Инвентарный слот: Диктофон, Старое удостоверение, Наручные часы, Лупа, Отмычки, Фото близкого человека
* Контакт: Бывший напарник, Хакер, Бармен, Адвокат

3. Игровой процесс
Игрок последовательно посещает локации театра:

Локация         | Что можно найти
----------------|----------------------
Закулисье    | Следы косметики, показания Колистилей
Гримёрка Евы       | Любовное письмо, признание Евы
Кабинет Верховского   | Билет Пожарского, документы Пожарского
Костюмерная      | Пятно крови, показания Зинаиды
Буфет    | Пузырёк от лекарств, чек из буфета
Склад реквизита | Орудие убийства

4. Механики
3 минуты на локацию — идёт обратный отсчёт
Прогресс-бары: Время (100%), Улики (6 настоящих), Характеристики (0-5)
Журнал улик — отображает все найденные улики
Контакт — даёт очки к характеристикам

5. Финал
Игрок называет убийцу. Настоящий убийца — Лика, помощница режиссёра.
Победа: назвал Лику + собрал 3+ настоящих улик
Поражение: не хватило улик или обвинил невиновного
